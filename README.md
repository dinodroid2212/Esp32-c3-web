# ESP32-C3 Web Server Project

## Deskripsi
Project ESP32-C3 untuk web server dengan berbagai fitur.

## Fitur
- [ ] Web server dengan WiFi
- [ ] Kontrol GPIO via web
- [ ] Sensor monitoring
- [ ] OTA Update

## Hardware
- Board: ESP32-C3 DevKit
- Sensor: (tambahkan jika ada)

## Instalasi
1. Clone repo ini
2. Buka dengan PlatformIO atau Arduino IDE
3. Atur WiFi credentials di kode
4. Upload ke ESP32-C3

## Penggunaan
- Buka serial monitor (115200 baud)
- Hubungkan ke WiFi
- Akses IP yang ditampilkan di browser

## Struktur File
- `Esp32c3web.ino` - Kode utama Arduino sketch
